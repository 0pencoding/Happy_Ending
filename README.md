## Branching 기법: git-flow

### <branch 종류>

- develop: 다음 출시 버전 개발 브랜치 (default branch)
- feature: 기능 개발 브랜치
  - Ex) feature/[종류]/[기능개발내용]
    [종류]: develop, fix, etc...
